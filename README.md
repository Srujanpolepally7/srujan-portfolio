# Portfolio — Srujankumar Polepally

A clean, fast, and responsive **static portfolio** ready to deploy on **Netlify** or any static host.

## Features
- Modern design with dark/light mode
- Responsive sections: Hero, Skills, Projects, Experience, Education, Certifications, Contact
- Netlify-ready contact form (no backend needed)
- Accessible, semantic HTML and keyboard-friendly navigation
- No build tools required — just upload

## Running locally
Open `index.html` in your browser, or start a simple dev server:
```bash
# Python 3
python -m http.server 8000
# navigate to http://localhost:8000
```

## Deploying to Netlify
1. Create a new site → **Deploy manually** → drag & drop the folder contents, or push to Git and connect.
2. Forms will work automatically via Netlify Forms (already configured).

## Customize
- Replace text in `index.html` (name, role, bio, links).
- Update colors in `styles.css` or switch default to light mode by adding `class="light"` on the `<html>` tag.
- Put your resume PDF at `assets/Srujankumar_Polepally_Resume.pdf` (or update the link).
- Replace images in `assets/` with your own.
